# ggplot_build method for tf_ggplot

ggplot_build method for tf_ggplot

## Usage

``` r
# S3 method for class 'tf_ggplot'
ggplot_build(plot, ...)
```

## Arguments

- plot:

  A tf_ggplot object

- ...:

  Additional arguments passed through from
  [`ggplot2::ggplot_build()`](https://ggplot2.tidyverse.org/reference/ggplot_build.html).

## Value

A built ggplot object (class `ggplot_built`).
