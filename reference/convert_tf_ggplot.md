# Convert tf_ggplot object to regular ggplot with transformed data

Convert tf_ggplot object to regular ggplot with transformed data

## Usage

``` r
convert_tf_ggplot(tf_plot, layer_mapping = aes())
```

## Arguments

- tf_plot:

  A tf_ggplot object

- layer_mapping:

  Additional aesthetic mapping from a layer

## Value

Regular ggplot object with transformed data
